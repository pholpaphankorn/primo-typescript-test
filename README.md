# primo-typescript-test

To ensure your project is ready to go, you can follow these final steps in your terminal:

1. **Initialize and Install:**

   ```bash
   yarn install
   ```

2. **Run the Logic:**
   To see the output of your `merge` function with sample data. You can edit input in `src/index.ts`:

   ```bash
   yarn dev
   ```

3. **Run the Tests:**
   To verify your solution against the edge cases (negative numbers, duplicates, and empty arrays):

   ```bash
   yarn test
   ```
